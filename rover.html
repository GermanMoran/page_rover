Basic Funcionality

The rover is made up of several modules

Gps Module

The GPS module allows the orientation and movement of the rover by inserting points of the globe in a web interface and tracing a route within the agricultural field.

Power Module

The energy module through the use of solar panels, supplies clean energy to the entire system.

Sensors Module

The sensor module allows to measure variables such as temperature, humidity and also has a flame detector.

Communications Module

The communications module allows connection to the database, where all the information captured by the sensors is stored.

General structure



The design of the rover parts was done using Solid Work software and 3D printed to ensure accurate measurements.

Solar Panels



This rover is working with renewable energy through the use of solar panels, which makes the rover autonomous during the entire operating time. With this we want to generate an environmental impact in the Colombian countryside.

Irrigation system



Our rover also has an irrigation system and uses the humidity variable to know if the land is in optimal conditions for sowing the corn seed, if the land is not in optimal conditions, irrigation system is carried out and the land is prepared.

Wi-Fi Module



The Wi-Fi module Esp2866 allows the connection with the Firebase database, and store all the variables taken by the sensors, and later use this information for other types of applications.

Fire variable



The rover also has a variable to measure fires and flames in the field because when the weather is extremely hot the corn fields can be seriously damaged, therefore this robot is able to find points in the field where exist flames and thus be able to save a crop in time.
